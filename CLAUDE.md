## Comandos

```bash
npm run dev      # dev server en localhost:3000
npm run build    # build de producción
npm run lint     # eslint
```

## Convenciones de commit

Micro-commits por área. Sin "Co-authored by Claude". Formato:

```
feat: descripción corta
chore: descripción corta
fix: descripción corta
```

## Lo que NO hacer

- No agregar comentarios obvios al código.
- No crear archivos de documentación intermedios.
- No envolver componentes en abstracciones innecesarias.
- No usar `any` en TypeScript.
